Step 1: Open a command line and navigate to the directory where you installed.

Step 2: Type -> npm install
		This will install all the dependencies.
		
Step 2: Type -> node srpb.js
		This will start the web server.
		
Step 3: Open file srpb.html in the client-example directory with your web browser of choice.
		Create - will allow you to create and entry
		Retrieve - will allow you to display all entries
		Update - will allow you to update an entry by ID
		Delete - will allow you to delete an entry by ID
